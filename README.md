# cheuler

Solutions to Project Euler problems in Chapel.

https://projecteuler.net

http://chapel.cray.com

SETUP
=====

Chapel (http://chapel.cray.com/download.html)

	brea install chapel

COMPILING SINGLE
 
    ./mk <x>.chpl

COMPILING ALL

    ./mk

RUN

All binary files are in the out directory

    ./out/<x>.chpl
